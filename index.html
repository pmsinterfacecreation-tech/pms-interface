PropAI · Real Estate Valuation Engine

Python · Flask · TensorFlow
The Problem

Most AI models die inside a local code notebook doing absolutely nothing for the real world. Real estate valuation is a slow, manual headache.

  • Humans can't easily calculate how local income, building age, & room counts interact on the fly without making mistakes.
  • Clients are stuck dealing with slow consultation calls.
  • Teams waste hours manually copy-pasting data.
The Solution

I solved this by building a clean, automated 3-tier ecosystem:

  • The Brain: A Python/Flask backend powered by a deep learning neural network (.h5) that calculates multi-variable property price predictions in milliseconds.
  • The UI: A clean, friction-free Tally form for clients.
  • The Pipeline: No manual paperwork. Direct, native data pipeline routing form submissions straight into a centralized Core Dashboard Spreadsheet.
The Results
  • Instant property price predictions
  • Data entry errors eliminated
  • Fully automated data pipeline

Key insight: True engineering isn't just about training a smart machine learning model. It's about building the automated data pipeline around it so it runs like a business while you sleep.
